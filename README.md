## PoC for fast prototyping PCBs using fiber laser

### Generated LightBurn files

* Etch
* Mask
* Stencil
* Placer

### Example Overview

* Red: copper
* Dark green: FR4
* Blue: soldermask cuts
* Light Green: soldermask marks (silkscreen)
* White: stencil cuts
